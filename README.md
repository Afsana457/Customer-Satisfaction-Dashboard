# Customer-Satisfaction-Dashboard
Power BI dashboard project using customer survey data
Customer Satisfaction Dashboard – Power BI Project

This Power BI dashboard analyzes customer satisfaction survey data from an airline company. It helps identify which factors (like seat comfort, cleanliness, and travel class) impact customer satisfaction.

## Dataset
The dataset includes survey responses with fields such as:
- Age, Gender, Travel Class
- Cleanliness Rating, Seat Comfort Rating
- Type of Travel, Satisfaction Level, etc.

## Dashboard Features
- Satisfaction by Age Group
- Cleanliness vs Satisfaction
- Seat Comfort vs Satisfaction
- Travel Class distribution
- Interactive slicers (Gender, Travel Type, etc.)
- Summary Cards for total & satisfied passengers

## Tools Used
- Power BI Desktop
- DAX for Age Group transformation

## File
- `survey_dashboard.pbix` – The Power BI dashboard file
